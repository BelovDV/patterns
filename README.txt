Работа закончилась после дедлайна, но только сейчас возникло ощущение, что результат достоин отправки.
Сама игра толком не начата, разработаны классы для графического оформления игры и для хранения данных. Готов класс для графических настроек.


Планируемый результат:
Вдохновление берется у различных игр серии Герои Меча и Магии, результат будет сильно отличаться, но его удобнее описывать в сравнении.
Планируется часть, где сражаются 2 героя (планируется человек-компьютер).
Карта гексогональная, бесконечная. Очередность ходов как в 5 героях. Параметры юнитов также схожи с 5 героями, но в одной клетке может быть только 1 юнит.


Сборка проекта:
windows, visual studio - на данный момент не поддерживается.
linux, gcc. Проверено в виртуальной lubuntu. Установить sfml ("sudo apt-get install libsfml-dev"). Собрать с помощью cmake.

В исходных файлах используется путь "../include/   " для подключения заголовков и "../data/   " для получения ресурсов.
