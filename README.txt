Сборка проекта:
Установить sfml ("sudo apt-get install libsfml-dev"). Собрать с помощью cmake.

После запуска в меню можно выбрать начало игры.
В новых меню выбирается класс, раса и юниты.
В самой игре почти бесконечное (невидимая граница есть, но до нее далеко) гексогональное поле.
ИИ не реализован, необходимо ходить за обоих игроков. При атаке перемещение непредсказуемо.
Когда у одной из сторон заканчиваются юниты игра выключается.
